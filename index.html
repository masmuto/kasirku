Integrasi Google Apps Script (GAS)

Untuk menyimpan data secara online di Google Spreadsheet, aktifkan fitur ini.

Panduan:
  1. Buka Google Sheets, ke Ekstensi > Apps Script.
  2. Copy kode di bawah ini dan paste ke Code.gs (Ganti semua kode yang ada).
  3. Klik Terapkan (Deploy) > Deployment Baru.
  4. Pilih "Aplikasi Web", set Akses ke "Siapa saja".
  5. Copy URL Web App dan paste di kolom atas.
